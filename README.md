gpt bot for lark
=

<h2>Start</h2>
<pre><code>
sh start.sh
</code></pre>

<h2>Config</h2>
<pre><code>
vim ~/conf/gpt_bot.yaml
</code></pre>

<pre><code>
lark:
    AppID: "cli_xxx"
    AppSecret: "xxx"
    EncryptKey: "xxx"
    VerificationToken: "xxx"
gpt:
    ApiKey: "sk-xxx"
</code></pre>
